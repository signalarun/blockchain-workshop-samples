## Setting up of development environment
 `
 $ sudo snap install node --classic
 ` 
 
 or
 
 `
 $ curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
 $ sudo apt-get install -y nodejs
 `
 
 `
 $ sudo npm install -g typescript
 `
## How to run ?
 `
 $ tsc first.ts`
 
 `
 $ node first.js
 `
